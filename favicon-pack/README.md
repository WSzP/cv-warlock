# Favicon Pack

Extract this package in the root of your web site project. If your site is http://www.example.com, you should be able to access a file named http://www.example.com/favicon.ico. Add this to header:

```html
<link rel="icon" type="image/png" href="/favicon-96x96.png" sizes="96x96" />
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
<link rel="shortcut icon" href="/favicon.ico" />
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png" />
<meta name="apple-mobile-web-app-title" content="CV Warlock" />
<link rel="manifest" href="/site.webmanifest" />
```
